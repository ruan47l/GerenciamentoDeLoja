Projeto responsivo que simula um sistema simples de loja, com funcionalidades básicas de verificação de login e cadastro. O sistema permite visualizar os usuários cadastrados, além de oferecer formas de adicionar, editar, excluir e procurar produtos.

Também possibilita realizar mais de uma venda por cliente, com quatro opções de pagamento. Após a finalização da venda, o sistema realiza o cadastro da venda.

Além disso, há a página de RVV, onde todas as vendas ficam registradas, bem como opções para gerenciar roubos ou vencimentos de produtos na loja.

Tecnologias usadas css,html, flask, python e javaScript
